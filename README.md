note
     -r [note name]: Read a note.
     -n [note name]: Write a new note.
     -l: Show all notes.
     -cname [note name] [new note name]: Change the note name.
     -d [note name]: Delete Note.
     -dAll: Delete all notes.
     -i: Shows some information about this tool.
     -h: Show this message.
